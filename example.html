🐛 BugDetector Pro

Demo de injeção vanilla em qualquer página. Teste os 3 métodos abaixo.

🔥 Método 1 — Script Tag Recomendado

Inclua no <head> da sua página:

<script src="packages/bug-detector/dist/bug-detector.iife.js"></script>
<script>
  BugDetector.init({
    shortcut: 'Ctrl+Shift+D',
    trigger: 'floating-button',
    branding: { primaryColor: '#06b6d4', position: 'bottom-right' }
  });
</script>

💉 Método 2 — Injeção via Console

Abra o DevTools (F12), cole no console:

BugDetector.inject(
  'https://raw.githack.com/Jhin1v9/bug-detector-pro/main/packages/bug-detector/dist/bug-detector.iife.js',
  { trigger: 'keyboard-shortcut', shortcut: 'Ctrl+Shift+D' }
);
